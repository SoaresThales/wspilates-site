## 🚀 O Projeto

O objetivo deste projeto é criar uma experiência de usuário (UX) focada em conversão, transmitindo o ambiente moderno e acolhedor do estúdio diretamente pelo navegador.

A arquitetura foi desenhada para ser **Data-Driven**. Todo o conteúdo (textos, links, cores, imagens do carrossel) é injetado através de um arquivo `data.json`. Isso permite que o visual e o conteúdo sejam desacoplados, facilitando a manutenção e a possível criação de templates para outros nichos.

## ✨ Principais Funcionalidades

- **Design Glassmorphism:** Menus flutuantes e botões com efeito de vidro opaco para uma estética "Premium".
- **Tipografia Editorial:** Uso inteligente de `clamp()` para responsividade fluida e `-webkit-hyphens` para justificação perfeita dos textos sem quebra de leitura.
- **Carrossel Interativo Completo:** Construído do zero em React com suporte a clique, miniaturas em tempo real e navegação fluida via teclado.
- **Smart Map:** Iframe do Google Maps nativo com cálculo de margens adaptativas via CSS.
- **Widget AI Native:** Ponto de captura flutuante para retenção de clientes.

## 💻 Como rodar localmente

```bash
# Clone este repositório
$ git clone [https://github.com/SEU-USUARIO/wspilates-site.git](https://github.com/SEU-USUARIO/wspilates-site.git)

# Acesse a pasta do projeto no terminal/cmd
$ cd wspilates-site

# Instale as dependências
$ npm install

# Execute a aplicação em modo de desenvolvimento
$ npm run dev
```
